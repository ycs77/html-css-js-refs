# HTML、CSS、JS 簡易速查表

## HTML

### 基本 Emmet

| Emmet                 | HTML                                               |
| --------------------- | -------------------------------------------------- |
| div                   | &lt;div&gt;&lt;/div&gt;                            |
| .my-class             | &lt;div class="my-class"&gt;&lt;/div&gt;           |
| button.my-class{按鈕} | &lt;button class="my-class"&gt;按鈕&lt;/button&gt; |
| #my-id                | &lt;div id="my-id"&gt;&lt;/div&gt;                 |
